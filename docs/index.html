Overview

MANTOU (馒头) is a modern, high-performance web framework that draws inspiration from Next.js and leverages the speed and efficiency of Bun. True to its namesake, the Chinese steamed bun, MANTOU is simple, lightweight, and designed to deliver powerful results with minimal complexity.

Key Features

  • File-Based Routing: Intuitive, Next.js-inspired routing structure.
  • Blazing Speed: Powered by Bun for unparalleled performance.
  • Automatic OpenAPI Documentation: Effortlessly generate API documentation.
  • Type-Safe API Routes: Enjoy robust type safety with minimal effort.
  • Next.js Patterns, Enhanced: Familiar paradigms with modern enhancements.

Why MANTOU?

MANTOU simplifies web development while delivering exceptional performance and developer experience. It’s a framework that’s fast, intuitive, and built to meet the demands of modern applications.
